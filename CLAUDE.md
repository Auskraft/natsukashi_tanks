# CLAUDE.md — Natsukashi Yoru — Танки

> Документ для продолжения проекта в новой сессии. Прочитай первым.

## Что это
Отдельная игра **«Танчики»** (Battle City-стиль) на **Flutter + Flame**, только
**Android**. `applicationId` **`com.auskraft.natsukashi_tanks`**, организация
`com.auskraft`. Отдельный продукт под брендом **Natsukashi Yoru** (у владельца
есть сборник из 14 мини-игр — проект `natsukashi_yoru`; эта игра вынесена из него
в **отдельный репозиторий** по решению владельца). Акцент — «сок»/дофамин и
удержание. Отображаемое имя приложения: «Natsukashi Yoru — Танки».

## ⚠️ Критично для агента
- **Собрать APK/AAB и `flutter run` агент НЕ может** — песочница инструментов
  блокирует Gradle. Это **не машина пользователя** (он собирает в Android Studio
  нормально). Агент верифицирует только `flutter analyze` + `flutter test`.
- **Коммитим по ходу** прямо в `main`, без напоминаний. В коммитах строка
  `Co-Authored-By: Claude Opus 4.8 (1M context) <noreply@anthropic.com>`.
- Нотифаер паузы в Flame-игре — **`isPaused`** (не `paused`: конфликт с
  `FlameGame.paused`).
- **Главный приоритет** — дофамин/«сок» + удержание.
- **Git:** ветка `main`, remote `origin` → `github.com/Auskraft/natsukashi_tanks`
  (репо создан и синхронизирован). Пуш — по просьбе владельца.

## Статус — что готово (фазы 0–7 + звук; остаётся публикация в RuStore)
- ✅ **Чистое ядро симуляции** (`lib/features/tanks/logic/`, только `dart:math`,
  инъекция `Random` → 100% тестируемо): суб-клеточная решётка 104×104, движение
  по полу-тайлу + выравнивание, пули со свип-субшагом (**анти-туннелинг**),
  поквадрантное разрушение кирпича, сталь/вода/лес/лёд, база-орёл, архетипы
  танков (basic/fast/power/armor/boss), тиры апгрейда игрока.
- ✅ **AI** (полу-случайный seek к базе/игроку + обход + огонь по лучу),
  **спавн-директор** (волны, лимит одновременных, точки сверху), вражеская
  стрельба, **бонусы** (звезда/граната/шлем/лопата/жизнь/таймер), инерция на льду.
- ✅ **Flame-слой** (`game/tanks_flame_game.dart`): гибрид-рендер (ретро-силуэты +
  неон-сок: взрывы/крошка/тряска/вспышки/попапы), нотифаеры HUD (+`isPaused`),
  пауза. **D-pad — единая зона-слайд + кнопка огня** (`core/components/dpad_control.dart`).
- ✅ **Витрина + 3 режима** (`app/`): **Кампания** (5 миров, 23 уровня, **боссы**
  с HP-баром, звёзды/анлоки), **Выживание** (эскалирующие волны, рекорд), **Вызов
  дня** (детерминированный по дате). Настройки (звук+хаптика), Оцените
  (рейтинг-гейт → RuStore/почта), Документы.
- ✅ **Формат уровней** (ASCII 13×13 + `level_parser` + `level_data` с мирами),
  темизация рендера по миру. Загрузчик `level_loader.buildLevel`.
- ✅ **Вход** `lib/main.dart` → домашняя витрина (сплэш → согласие → витрина).
- ✅ **60 тестов** зелёные, `flutter analyze` чисто.
- ✅ **Аудио-система (Фаза 6, код):** `core/audio/AudioManager` — loop-музыка,
  SFX через пулы (`AudioPool` на эффект — мгновенно, без рассинхрона), дакинг,
  громкость/мьют (персист), прелоад; `MusicTrack`/
  `SfxEvent` разведены по Flame-слою (на исходы `TankStep`), музыка меню/боя/
  босса, стингеры победы/поражения, музыка+тап в витрине, экран настроек звука.
  **Безопасно без файлов и в тестах** (гард `FLUTTER_TEST` + try/catch = no-op).
  Клипы (3 музыки + 17 SFX) **сгенерированы** (`tool/gen_audio.py` — sfxr-стиль +
  мини-трекер, WAV, CC0/собственность) — игра звучит (плейтест: рассинхрон SFX
  устранён пулами `AudioPool`).
  Манифест / как заменить / CC0-источники — [`docs/AUDIO.md`](docs/AUDIO.md).
- ✅ **Плейтест-фиксы:** боты обходят препятствия + фикс залипания при высокой
  герцовке (флаг `Tank.blocked` = реально упёрся, а не «не сдвинулся за кадр»);
  управление = **слайд-пад** (скольжение пальцем, без мёртвых зон между кнопками);
  выстрел сносит **половину кирпича** за раз; мягче баланс; корректный выход
  «В меню» (без чёрного экрана).

## Архитектура (4 слоя на игру, эталон — `features/tanks/`)
- `logic/*` — **чистая логика** (Random-инъекция), `step(dt, PlayerIntent) →
  TankStep` (структура исходов для «сока»). Тесты: `test/tanks_*_test.dart`
  (русские названия, `Random(seed)`, проверки детерминизма).
- `game/tanks_flame_game.dart` — `FlameGame`: фазы ready/running/dead, нотифаеры,
  `_active = running && !isPaused`, `update→step`, рендер + «сок».
- `ui/tanks_overlays.dart` — HUD из `core/components/overlay_kit.dart`.
- `app/tanks_game_screen.dart` — host-экран (GameWidget + D-pad + оверлеи +
  запись рекорда/стрика в `GameStorage`).
- `core/` — переиспользуемое (скопировано из сборника, теперь живёт здесь
  независимо): `theme`, `storage/GameStorage`, `feedback/Haptics`,
  `components` (overlay_kit, dpad_control, game_scaffold), `legal`, `audio`
  (фасад-заглушка над `flame_audio`), `rating`, `achievements` (заглушка).
