## Короткое описание (до ~80 символов)
Ретро-танки в духе Battle City: кампания, боссы, выживание и вызов дня.

## Полное описание
«Natsukashi Yoru Tanks» — олдскульная аркада в духе классических танковых баталий 8-бит эпохи,
но с современным «соком»: сочные взрывы, тряска экрана, неон-частицы,
вибро-отклик и драйвовый чиптюн-саундтрек.

Защищай базу-орла, кроши кирпичные стены по кускам, прячься в лесу, переигрывай
врагов на скользком льду и зачищай волну за волной. Управление — удобный
слайд-пад под большой палец: ведёшь — едешь, жмёшь — стреляешь.

🎯 **Режимы:**
• **Кампания** — 5 миров, 23 уровня, боссы с полосой HP, звёзды и открытие новых уровней.
• **Выживание** — бесконечные нарастающие волны, борись за рекорд.
• **Вызов дня** — один расклад на всех, новый каждый день.

⚙️ **Что внутри:**
• Разрушаемый мир: кирпич (по кускам), сталь, вода, лес-укрытие, лёд с инерцией.
• Бонусы: звезда-апгрейд, граната, шлем-щит, лопата-броня базы, +жизнь, заморозка.
• Умные враги: разные типы танков, обход препятствий, огонь по линии.
• Чиптюн-музыка и звуки, тонкая вибрация, поддержка экранов 90/120 Гц.

📴 Полностью офлайн. Без рекламы. Без интернета. Без доната.

## Чеклист публикации
1. **Подпись:** создать `android/key.properties` + `upload-keystore.jks` (или
   переиспользовать keystore сборника) — см. `android/key.properties.example`.
   `build.gradle.kts` уже читает их для релиза.
2. **Сборка:** `flutter build appbundle --release` (или `apk` — оба подойдут).
3. **Скриншоты:** несколько экранов — меню, бой, босс, выживание.
4. **(Опц.) Иконка/сплэш «танков»** — сейчас переиспользуется иконка сборника
   (`assets/icon/`). Для отдельного бренда — свой арт + `dart run
   flutter_launcher_icons` / `flutter_native_splash:create`.
5. Залить в RuStore, заполнить карточку этим описанием, выбрать категорию/возраст.
