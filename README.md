# Natsukashi Yoru — Танки

Отдельная игра **«Танчики»** (Battle City-стиль) на **Flutter + Flame**, Android.
Часть бренда **Natsukashi Yoru** (есть ещё сборник из 14 мини-игр — проект
`natsukashi_yoru`). `applicationId` — `com.auskraft.natsukashi_tanks`.

## Что есть
- **Кампания** — 5 миров (Двор / Завод / Река / Роща / Полигон), 23 уровня,
  **боссы** с HP-баром, звёзды (пройти / время / без потери жизни) и анлоки.
- **Выживание** — эскалирующие волны, погоня за рекордом.
- **Вызов дня** — детерминированная по дате карта (один расклад на всех).
- Чистое тестируемое ядро симуляции (суб-клеточная решётка, анти-туннелинг пуль,
  разрушаемый терреин), гибрид-рендер со «соком», слайд-D-pad.

## Запуск
```bash
flutter pub get
flutter run        # вход lib/main.dart
flutter analyze    # должно быть «No issues found!»
flutter test       # 58 тестов
```

Сборка APK/AAB — в Android Studio (релизная подпись через `android/key.properties`,
он в `.gitignore`).
